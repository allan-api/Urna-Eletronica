## Inicialização do Projeto

1. **Clone o Repositório:**

```bash
  git clone https://github.com/allan-api/Urna-Eletronica.git
```

2. **Crie um Ambiente Virtual (Opcional, mas Recomendado)**

```bash
  python -m venv venv
```

3. **Execute o projeto**

```bash
  python main.py
```
